## Basic Image Usage

### Method 1: Standard Markdown Syntax

The simplest way to add images is using standard markdown syntax:

```markdown
![Alt text](/static/images/your-image.jpg)
```

**Example:**

```markdown
![Mountains](/static/images/canada/mountains.jpg)
```

**Rendered Result:**

![Mountains](/static/images/canada/mountains.jpg)

### Method 2: Direct Next.js Image Component

For more control, use the Next.js Image component directly in MDX:

```jsx
<Image alt="Description of image" src="/static/images/your-image.jpg" width={800} height={600} />
```

## Frontmatter Image Metadata

You can specify images in the frontmatter for social media previews and SEO:

```yaml
---
title: 'Your Blog Post'
date: '2024-01-28'
tags: ['tutorial']
draft: false
summary: 'Blog post description'
images: ['/static/images/your-image.jpg']
---
```

Multiple images can be specified:

```yaml
images: ['/static/images/image1.jpg', '/static/images/image2.jpg']
```

## Image Optimization Features

The Next.js Image component provides:

- **Automatic WebP conversion** - ~30% smaller file sizes
- **Responsive images** - Correct size served based on viewport
- **Lazy loading** - Images load as you scroll
- **Layout shift prevention** - Avoids Cumulative Layout Shift (CLS)
- **On-demand optimization** - No increase in build time

## Troubleshooting

### Images Not Loading?

- Check file path: `/static/images/your-image.jpg`
- Ensure image is in `/public/static/images/` directory
- Verify file name spelling and case sensitivity

### Images Not Optimized?

- Only local images are optimized
- External URLs won't use Next.js Image component
- Ensure `imgToJsx` is enabled in `lib/mdx.js`

### Layout Issues?

- Use responsive grid classes
- Test on different screen sizes
- Check Tailwind CSS class syntax

## External Images

For external images (from CDNs, Unsplash, etc.), use standard HTML img tags:

```html
<img src="https://example.com/image.jpg" alt="Description" className="rounded-lg" />
```

Note: External images won't get Next.js optimization benefits.
